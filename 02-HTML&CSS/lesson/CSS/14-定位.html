相对定位：relative定位
  1. 参照物：自身原来所在位置
  2. 存在DOM流中，占据原先的空间
  3. 必须配合toli/bottom/left/right四个方位才能生效
  4. 上下选一个，左右选一个
  5. 微调某一个元素的位置，常被用来作为绝对定位元素的参照物。

sticky定位
  1. 粘性定位，该定位基于用户滚动的位置。
  2. 参照物：离它最近的一个拥有“滚动机制”的祖先上（当该祖先的overflow 是 hidden, scroll, auto, 或 overlay时）和containing block (nearest block-level ancestor) 最近块级祖先。
  3. 它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。
  4. Internet Explorer, Edge 15 及更早 IE 版本不支持 sticky 定位。 Safari 需要使用 -webkit- prefix
绝对定位：absolute定位
  1. 1、参照物：最靠近的已定位（fixed/relative/absolute）祖先元素，如果没有已定位的祖先元素，那么它的位置相对于html或者body定位;
  2. 2、脱离DOM流，不占据空间。
  3. 3、必须配合top/bottom/left/right四个方位才能生效
  4. 4、上下选一个，左右选一个
  5. 5、常用作动画结构

z-index
  1. 两个定位元素，默认条件下，后面盖住前面
  2. 同一个参照物，z-index数值越大，越靠前
  3. 只对定位（relative/fixed/absolute/sticky）元素有效，单独使用无效。

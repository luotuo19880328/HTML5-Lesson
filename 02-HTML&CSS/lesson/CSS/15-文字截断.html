新华社里约热内卢12月23日电(记者陈威华赵焱)新年即将到来，巴西新冠疫情依然十分严峻。主要由华商组成的巴西里约华人联谊会(以下简称华联会)近日面向当地多家机构举行“赠人玫瑰手留余香”慈善物资捐赠活动，得到巴西各级政府和民众的赞许。 据华联会秘书长苏子梁介绍，华联会每年都举行针对当地孤儿院、敬老院和福利院等机构的捐助活动，以回馈当地社会。今年由于疫情严重，巴西失业率上升，底层民众生计艰难。华联会购买了一批专供老人和儿童的食品以及防疫物资，捐给由当地政府推荐的4家机构。
